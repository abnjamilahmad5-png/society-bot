## 🛠️ خطوات التثبيت

### المتطلبات
- Python 3.10+
- pip (مدير الحزم)
- حساب Discord Developer

### الخطوة 1: استنساخ أو تنزيل المشروع
```bash
git clone https://github.com/your-repo/society-bot.git
cd society-bot
```

### الخطوة 2: تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### الخطوة 3: إعداد config.json
1. افتح ملف `config.json`
2. أضف **TOKEN** البوت الخاص بك من [Discord Developer Portal](https://discord.com/developers/applications)
3. أضف **GUILD_ID** (ID السيرفر للاختبار)
4. قم بتعديل باقي الـ IDs حسب السيرفر الخاص بك

**مثال:**
```json
{
  "TOKEN": "YOUR_BOT_TOKEN_HERE",
  "PREFIX": "!",
  "GUILD_ID": 123456789,
  "ADMIN_ROLE": 987654321,
  ...
}
```

### الخطوة 4: تشغيل البوت
```bash
python main.py
```

عند التشغيل الناجح سترى:
```
==================================================
✓ تم تسجيل دخول البوت: 𝑆𝑜𝑐𝑖𝑒𝑡𝑦
✓ ID البوت: 1234567890
✓ عدد السيرفرات: 1
✓ عدد المستخدمين: 50
==================================================
```

## 🔧 الإعدادات المتقدمة

### تخصيص الألوان
عدّل `ColorManager` في `main.py`:
```python
class ColorManager:
    PRIMARY = 0x36017F      # اللون الأساسي
    SUCCESS = 0x00AA00      # أخضر
    ERROR = 0xFF0000        # أحمر
    WARNING = 0xFFAA00      # برتقالي
```

### تغيير الـ Prefix
غيّر في `config.json`:
```json
"PREFIX": "!"
```

### تفعيل/تعطيل الأنظمة
في `config.json`:
```json
"ENABLED_SYSTEMS": {
    "tickets": true,
    "protection": true,
    ...
}
```
