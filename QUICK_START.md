## 🎯 الخطوات الثلاث الأولى

### 1️⃣ تثبيت المكتبات
```bash

```

### 2️⃣ إضافة TOKEN
افتح `config.json` وأضف:
```json

"GUILD_ID": YOUR_SERVER_ID
```

### 3️⃣ تشغيل البوت
```bash
python main.py
```
